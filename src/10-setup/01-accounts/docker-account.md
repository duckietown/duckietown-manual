# Docker Account

[Docker](https://www.docker.com/) is a software technology that enables virtualization of operating systems, and "shipping" software in packages known as "containers". This containerization makes it easy to "download and run" complex software packages.

Duckietown leverages containerization to ensure software portability and reproducibility, and in particular [DockerHub](https://hub.docker.com/) to distribute it.

Most procedures in Duckietown use `docker` operations behind the hood. That is why Duckietown users need to set up a Docker account, and (later) synchronize it with the Duckietown Shell.

(dt-account-dockerhub)=
## Create a DockerHub account

If you do not have one already, create your free DockerHub account now:

* [Signup for a DockerHub account](https://hub.docker.com/signup).
