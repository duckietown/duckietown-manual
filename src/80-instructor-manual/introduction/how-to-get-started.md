# How to Get Started

Ideally you are reading this book in advance of starting a class, so you have time to plan and prepare. 

You might be considering teaching a complete Duckietown class, or integrating [the Duckietown resources](instructor-manual-resources-overview) in a preexisting class.

If this is your case, proceed with the [next sections](how-to-use) in the order presented. You can use this manual to:

1. Gain insights on the [technical resources](instructor-manual-resources-overview) offered by Duckietown;
2. Decide the scope of your class by reviewing [sample curricula](graduate-sample-curricula) and [classroom slides](lecture-slides);
3. Read the [course design considerations](considerations-prerequisites) and understand what is needed or nice to have in setting up a class. 

```{note}
If the beginning of your class is instead imminent, go directly to:

* [Teaching with Duckietown - Quick Start Guide](quickstart-intro).
```
