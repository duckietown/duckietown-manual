## Running Demos on Virtual Duckietown Robots

To run a demo in the Duckiematrix:

1. Run the Duckiematrix.
2. Attach a Duckietown robot to a Duckiematrix entity.
3. Run the demo as you would using a physical Duckietown robot (e.g., [](duckiebot-demo-lf)).
