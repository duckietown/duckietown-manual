# How to start developing for Duckietown

The Beginner section of the Developer Manual is designed to help new Duckietown contributors get started with development. Here you'll find step-by-step guides on setting up projects, working with ROS, creating Python libraries, understanding the code structure, and documenting your work. Whether you're new to robotics or just new to Duckietown, this section will guide you through the essential tools and workflows.
