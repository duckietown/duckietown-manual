# How to get Help 

Getting (and providing) help is part of the learning process.

Unless you are an instructor with an [instructor pack subscription](https://get.duckietown.com/products/instructor-pack-subscription?variant=44201386180783), Duckietown support is community-driven. This means technical support is provided on a best-effort basis, without guarantees of timeliness. 

Follow these steps if you run in any technical issue: 

```{admonition} How to get help
:class: tip
1. Check the troubleshooting section at the end of each instruction manual page (when applicable)
3. Check the [Duckietown Archives](https://stackoverflowteams.com/c/duckietown/questions) if your problem has already been solved ([How to sign up to the Duckietown Archives](setup-account-stack-overflow))
4. Create a post on the Duckietown Archives
5. Search [the Duckietown community](https://duckietown.slack.com/) ([Join the Duckietown Community on Slack](setup-account-slack)) 
6. Link your Duckietown Archives question to the appropriate #help channel.
```

When asking for help make sure to provide sufficient information for others to reproduce your error. I.e., specify what:

- OS are you using
- instructions are you following
- error you are getting.

```{tip} 
Saying "it doesn't work" is rarely helpful, as it could imply too many things.
```
